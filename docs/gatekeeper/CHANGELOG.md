# 문지기봇 변경 이력

## [2026-01-07] 문서 정리 및 거버넌스 구축

### 변경 사항
- 프로젝트 전체 문서(.md) 정리 및 분류
- Gatekeeper 관련 문서를 `docs/gatekeeper/` 디렉토리로 이동
- 문서 거버넌스 체계 구축

### 이동된 문서
- `DESIGN_REVIEW.md`: 설계 검토 결과
- `DESIGN_INTENT_REFLECTION.md`: 설계 의도 코드 반영
- `IMPLEMENTATION_SUMMARY.md`: 구현 완료 요약
- `SECONDARY_FILTER_IMPLEMENTATION.md`: 2차 필터 구현
- `COMPLIANCE_CHECK.md`: 구현 완료 기준 검증
- `ROLE_COMPLIANCE_AUDIT.md`: 역할 및 범위 검증
- `OUTPUT_CONTRACT.md`: 출력 데이터 계약
- `OUTPUT_FORMAT_VERIFICATION.md`: 출력 형식 검증
- `WATCHLIST_FORMAT_SPEC.md`: Watchlist 형식 명세
- `INPUT_FORMAT.md`: 입력 포맷 설명

### 목적
- 문서 체계화 및 가독성 향상
- 공식 문서(Design, Spec) 명확히 표시
- 문서 관리 효율성 향상

### 문서 분류 결과
- **설계 문서**: `DESIGN_REVIEW.md`, `DESIGN_INTENT_REFLECTION.md`
- **구현 문서**: `IMPLEMENTATION_SUMMARY.md`, `SECONDARY_FILTER_IMPLEMENTATION.md`
- **검증 문서**: `COMPLIANCE_CHECK.md`, `ROLE_COMPLIANCE_AUDIT.md`, `OUTPUT_FORMAT_VERIFICATION.md`
- **계약 문서**: `OUTPUT_CONTRACT.md`, `WATCHLIST_FORMAT_SPEC.md`, `INPUT_FORMAT.md`
